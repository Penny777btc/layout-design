layout-design · 安装与使用说明
================================

这是什么
--------
一个给「视频剪辑 / 花字 / 字幕 / 封面 / 图形」用的版面排版判断技能（Claude skill）。
它把日本平面设计的版面法则，翻译成「这版排版好不好、该怎么改」的可执行判断 + 质检清单。
完整说明见 README.md（可用浏览器或文本编辑器打开），里面有 10 个法则的「使用前/使用后」对比图（images 文件夹）。

配套技能：shot-language（管镜头里的画面构图）。一套镜头，一套版面。


用法一：只想学版面设计，不用 AI
-------------------------------
直接看 README.md 和 images 里的对比图即可，无需安装。
SKILL.md 末尾有「七条版面体检」清单，可拿去给自己的封面/花字做体检。


用法二：当 Claude Code 技能来用
-------------------------------
1. 解压本压缩包，得到 layout-design 文件夹。
2. 把整个 layout-design 文件夹放到下面任一位置：

   · 只在某个项目里用：
       你的项目目录/.claude/skills/layout-design/

   · 全局都能用：
       macOS / Linux :  ~/.claude/skills/layout-design/
       Windows       :  C:\Users\你的用户名\.claude\skills\layout-design\

3. 重新打开 Claude Code。之后你做花字/字幕/封面/图形/质检时，它会自动参考；
   也可以直接说："用 layout-design 看看这个封面排版有什么问题"。


文件说明
--------

来源：《日本版面的法則》(gaatii 光體)
