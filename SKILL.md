# 版面排版与平面设计（判断库）

来源：《日本版面的法則》(gaatii 光體)。把书里 110 个日本设计案例的版面法则，
转译为**本管线的图形排版判断**——花字、字幕、封面、缩略图、Remotion 合成图形。
深度案例与原理见 [reference.md](reference.md)；本文件只放**何时用、怎么判断、怎么验收**。

## 核心前提（先记住这一条）

版面的好坏 = **主次清不清楚 + 对不对齐 + 透不透气**。

> "高级感"不是加装饰，而是 —— 定一个视觉核心 → 用字级/位置/留白分出主次 → 上网格对齐 → 配色克制。

跟 [[shot-language]] 是一对：镜头语言管"画面里看什么"，版面排版管"文字图形怎么摆"。

## 何时调用 / 接入点

| 场景 | 本 skill 提供的判断 |
|---|---|
| 阶段 5 **花字 / 字幕** | 大中小字级分主次、直排/横排、文字别挡主体、标题字呼应主题 |
| **封面 / 缩略图**（社媒） | 一个视觉核心(大标题 or 图)+ 留白；主色 + 少量信号/对比色；标题字呼应内容 |
| 阶段 5 **Remotion 图形 / 动效卡片** | 网格对齐、重复/渐变/对比组合、配色克制、系列版式统一 |
| **海报 / 宣传图** | 五大模块全套 |
| 图形质检 | 跑下方「七条版面体检」 |

## 五大模块速查

**① 平面构成**（怎么摆元素）
- 八种构成形式备选：重复 / 近似 / 渐变 / 放射 / 轴向 / 密集 / 对比 / 节奏。
- 最常用组合：**重复 + 大小渐变/对比**(立刻有主次和韵律)。
- 先用**网格系统**分割版面再放元素——对齐是"整洁感"的来源。
- 对角线构图＝动感；中轴对称＝稳定。

**② 文字排版**（信息怎么分级）
- **大中小三级字号分主次**：最重要的信息最大，次要的缩小退后。一屏只一个"最大"。
- 直排 / 横排可混用增层次，但要有理由(顺应主体方向/营造对比)，别为混而混。
- 标题字可定制，**呼应主题**(融合主体笔画、家徽、插画元素)——比换字库更高级。
- 辅助手段：文绕图、底线/色条强调、字符符号当引导点、文字沿图形轮廓排。
- 文字轻微倾斜＝动感(需配合主题，别用在要稳重的内容)。

**③ 色彩**（怎么定调）
- **主色呼应主题情绪**(暖=活力/亲近，冷=安静/理性)，先定主色再配。
- 重点靠**少量高饱和/信号色/对比色**(互补、冷暖、明暗、数量对比)；大面积 vs 一点＝最高级的对比。
- CMYK 纯色块＝利落现代；渐变＝层次/梦幻。
- 特色印刷(荧光/金/银/磨砂)＝质感——数字版＝发光、金属质感、颗粒叠加。

**④ 留白**（怎么聚焦）
- **大面积留白＝聚焦主体 + 传达"静/素/高级"**。来自水墨、侘寂、花道的东方美学。
- 边缘留白让版面透气；**别填满**——满＝廉价、乱。

**⑤ 配图**（图怎么用）
- 图(插画/摄影)常是版面**视觉核心**，先定图再排字。
- 手写字 / 手绘要和插画**风格呼应统一**；摄影则求干净。
- 近大远小＝纵深；文字可绕图/沿轮廓排，让图文一体。
- 系列(多张)要**版式统一 + 局部变化**(频道一致性)。

## 七条版面体检（图形质检时逐条过）

- [ ] **没主次**：信息全一样大 → 用大中小字级 + 位置分出唯一核心。
- [ ] **没对齐**：元素乱摆 → 上网格、对齐边线。
- [ ] **太满**：填满无留白 → 删/缩次要元素，留白聚焦主体。
- [ ] **配色脏/多**：颜色超过主色+2 → 收到主色 + 少量重点色。
- [ ] **字体不搭**：标题字与主题/情绪不符 → 换呼应主题的字体或定制。
- [ ] **图文打架**：文字挡住主体、方向混乱 → 文字让位/绕图、统一主导方向。
- [ ] **系列不统一**：多张风格各异 → 固定版式骨架，只换内容/配色。

## 关注度 vs 可信度（同 [[shot-language]]）

越花哨(怪字体、强对比、满版)＝越吸睛但越"廉价/不可信"；越克制(留白、网格、标准字)＝越高级可信。
教程/产品类＝克制优先；Hook/封面＝可适度花哨博点击。按调性定档。
